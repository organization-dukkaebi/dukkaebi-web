## 이슈

---

1. 코스 개발하다가 끝나서 코스 부분은 테스트가 덜 되어있어서 api 관련 오류가 있을수 있음
2. 각 기능마다 다른 사람이 만들어서 일부 코드 스타일이 다른 부분이 있을수 있음 (마지막에 전반적으로 수정 하기는 했는데 정확할지는 알수 없음)
3. 문제 쪽에서 전반적인 테스트를 진행한 적이 있으나, 실제 이 서비스를 가지고 대회를 한 적은 없기에, 만일
   대회시에 이 서비스 사용시 완전히 다른 사람이나 혹은 여러가지 (서버 부하 테스트, 프론트 api 확인 등등)
   해보고 대회를 진행 하는걸 추천함
4. 이 프로젝트는 관리자랑 학생이 나누어져 있으므로 따로 각각 관리가 필요함, 코드 구조나 개발 방법,
   기술 스택이 다르지는 않으나 확인이 필요함
5. 배포는 vercel로 되어있고 권한 필요하면 “차동규” 한테 연락
